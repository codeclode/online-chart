## 如何让程序运行起来
<Callout type="error" emoji="😭">
  数据库——可能是最麻烦的地方。
</Callout>
为了让mongodb支持事务，prisma要求如果使用mongodb作为数据库，那么需要启动三个mongodb作为主、从、投票。
因此如果想要启动，必须先了解如何启动三个mongodb。

<Link style={{color:"#6cf"}} target="_blank" href="https://blog.csdn.net/weixin_52851967/article/details/126860698">复制集启动攻略</Link>

<Callout type="warning" emoji="✨">
  依赖与prisma生成
</Callout>
```shell
npm i
#下载依赖
npx prisma init --datasource-provider sqlite 
#初始化prisma,请记得在此之前修改.env中DATABASE_URL为您自己的mongodb路径
prisma generate
#初始化prisma,请记得在此之前修改.env中DATABASE_URL为您自己的mongodb路径
```
如果prisma出现问题，那么请前往
<Link style={{color:"#6cf"}} target="_blank" href="
https://www.prisma.io/docs/getting-started/setup-prisma/start-from-scratch/mongodb-typescript-mongodb">prisma+mongodb启动方案</Link>
查看命令顺序(笔者不敢确保上面的命令可以在你的电脑正确运行)

<Callout type="info" emoji="😎">
  成功了不
</Callout>
```shell
npm run dev
```
如果到了这一步，应该就没有问题了。
